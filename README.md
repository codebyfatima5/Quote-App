# Multi-Language Quote App

A responsive **Quote Generator web app** with English & Urdu support, dark/light mode, save/copy features, and smooth animations. Perfect for showcasing frontend skills or building a small SaaS project.

---

## Features

- **Multi-language support:** English & Urdu quotes  
- **Dark/Light Mode:** Automatic text color changes for readability  
- **Save & Delete Quotes:** Store favorite quotes locally  
- **Copy Quotes:** Easily copy quotes to clipboard  
- **Mobile Responsive:** Works perfectly on all devices  
- **Smooth Animations:** Fade effects for quotes and interactions  
- **API Fallback:** Loads fallback quotes if API fails  

---

## Tech Stack

- HTML  
- CSS (Variables + Transitions)  
- JavaScript (Fetch API + localStorage)
